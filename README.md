## Mini HTTP request sender
This tools sends HTTP request in GET method and gives the respond (header or text) back to you

## How it works:
- download this repo using terminal or download the folder here
- go to the directory were the main.py of project exists
- run main.py
- write the target URL, remember to write http:// or https:// at first
- choose if you want the header or all of the response, remember to only write HEADER or ALL
